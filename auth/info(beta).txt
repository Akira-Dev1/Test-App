***MONGO***

mongosh = Mongo Shell, это CLI-клиент для MongoDB

Аналоги:
 - psql для PostgreSQL
 - mysql для MySQL

use authdb
db.users.find().pretty() - Тут я:
 - Подключился к MongoDB
 - Выбрал базу authdb
 - Посмотрел коллекцию users
 - Увидел реальные документы

Для чего нужен mongosh:
 - ручная проверка данных
 - отладка
 - тестирование схем
 - диагностика проблем

В проде mongosh не пользуются для логики, только для администрирования.


***ЗАПРОСЫ***

TCP-соединение
 └── HTTP-сессия
      ├── Строка статуса (200 OK)
      ├── Заголовки (Headers)
      └── Тело ответа (Body)  

В одном tcp-соединении может быть несколько HTTP-запросов, но все они идти будут строго по очереди, не параллельно (Если HTTP 1.1), если 2 или 3 - то могут
